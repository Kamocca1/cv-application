üîß How It Works

PDF Export Process:

  1. Creates a temporary DOM container with A4 dimensions
  2. Clones the CV preview content with optimized styling
  3. Uses html2canvas to capture the content as an image
  4. Embeds the image in a PDF using jsPDF
  5. Downloads the file with a generated filename

Word Export Process:

  1. Extracts CV data from the application state
  2. Creates structured document sections using docx library
  3. Formats text with appropriate headings, fonts, and spacing
  4. Generates a .docx file and triggers download
